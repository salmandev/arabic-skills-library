## الهدف

إتقان أساسيات اختبار الاختراق (Penetration Testing).

## مراحل الاختبار

### ١. الاستطلاع (Reconnaissance)

```
□ جمع معلومات مفتوحة (OSINT)
□ فحص شبكي
□ تحديد أهداف
```

### ٢. المسح (Scanning)

```
□ فحص منافذ (Nmap)
□ كشف خدمات
□ كشف ثغرات
```

### ٣. الاستغلال (Exploitation)

```
□ استغلال ثغرات
□ الحصول على وصول
□ رفع صلاحيات
```

### ٤. الحفاظ على الوصول

```
□ تثبيت أبواب خلفية
□ إنشاء مستخدمين
□ إخفاء آثار
```

### ٥. التغطية والإبلاغ

```
□ توثيق نتائج
□ تقييم مخاطر
□ توصيات علاج
```

## أدوات شائعة

| الأداة | الاستخدام |
|--------|----------|
| **Kali Linux** | نظام اختراق |
| **Metasploit** | استغلال ثغرات |
| **Burp Suite** | اختبار ويب |
| **Wireshark** | تحليل حزم |
| **John the Ripper** | كسر كلمات مرور |

## أفضل الممارسات

✅ احصل على إذن كتابي
✅ وثّق كل شيء
✅ التزم بالنطاق
✅ قدّم توصيات عملية

❌ لا تختبر بدون إذن
❌ لا تتجاوز النطاق
❌ لا تكشف معلومات حساسة
