## الهدف

إتقان أساسيات إنتاج الأفلام والفيديو.

## مراحل إنتاج

### ١. ما قبل إنتاج

```
□ فكرة وقصة
□ سيناريو
□ ميزانية
□ طاقم عمل
□ مواقع تصوير
□ جدول تصوير
```

### ٢. إنتاج

```
✓ تصوير
✓ إضاءة
✓ صوت
✓ إخراج
✓ تمثيل
```

### ٣. بعد إنتاج

```
✓ مونتاج
✓ مؤثرات بصرية
✓ صوت وموسيقى
✓ تلوين
✓ تصدير نهائي
```

## أنواع أفلام

| النوع | الوصف |
|-------|-------|
| **روائي** | قصة خيالية |
| **وثائقي** | واقع وحقيقة |
| **قصير** | أقل من ٤٠ دقيقة |
| **طويل** | أكثر من ٤٠ دقيقة |
| **متحرك** | Animation |
