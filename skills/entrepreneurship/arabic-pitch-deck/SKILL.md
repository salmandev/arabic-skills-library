## الهدف

إتقان أساسيات إنشاء عرض تقديمي جذاب للمستثمرين (Pitch Deck).

## شرائح أساسية

### ١. مشكلة

```
✓ ما المشكلة؟
✓ من يتأثر؟
✓ حجم الألم
```

### ٢. حل

```
✓ ما الحل؟
✓ كيف يعمل؟
✓ لماذا أفضل؟
```

### ٣. سوق

```
✓ حجم سوق
✓ نمو متوقع
✓ شريحة مستهدفة
```

### ٤. نموذج عمل

```
✓ كيف تكسب مال؟
✓ مصادر إيرادات
✓ هوامش ربح
```

### ٥. فريق

```
✓ مؤسسون
✓ خبرات
✓ مستشارون
```

### ٦. طلب

```
✓ مبلغ مطلوب
✓ استخدام أموال
✓ معالم مستقبلية
```
