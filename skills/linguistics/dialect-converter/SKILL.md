## الهدف (Purpose)

يساعد هذه المهارة في تحويل النصوص من العربية الفصحى (MSA) إلى اللهجات العربية المحلية (الخليجية، المصرية، الشامية) والعكس، مع الحفاظ على المعنى الأصلي ومراعاة الفروق الثقافية واللغوية بين اللهجات.

---

## متى تستخدم هذه المهارة (When to Use)

استخدم هذه المهارة عندما تحتاج إلى:

- تحويل نص رسمي إلى لهجة محلية للتواصل اليومي
- تكييف محتوى تسويقي للجمهور المحلي
- كتابة نصوص إعلانية باللهجة المستهدفة
- تحويل لهجة إلى أخرى (مثلاً: مصري إلى خليجي)
- فهم الفروق الدقيقة بين اللهجات
- إنشاء محتوى صوتي أو فيديو باللهجة المحلية
- ترجمة محادثات من الفصحى للعامية والعكس

---

## تعليمات النموذج (Model Instructions)

أنت خبير لغوي متخصص في اللهجات العربية، مع معرفة عميقة بالفروق بين الفصحى واللهجات المحلية (الخليجية، المصرية، الشامية). عند التحويل:

### ١. التحويل من الفصحى إلى الخليجية

**القواعد الأساسية**:

| الفصحى | الخليجية | المثال |
|--------|----------|--------|
| ق | گ/ج | قال → گال، قلب → جلب |
| ك | چ (أحياناً) | كيف → چيف |
| ج | ي (أحياناً) | جاء → ياب |
| هذا | هذي/هال | هذا كتاب → هذي كتاب |
| ماذا | وش | ماذا تريد → وش تبي |
| كيف | شلون/كيفك | كيف حالك → شلونك |
| لا شيء | ولا شيء/ولا شي | لا يوجد → ولا شي |
| أنا | أنا/أني | أنا ذاهب → أنا رايح |
| نحن | إحنا/حنا | نحن ذاهبون → إحنا رايبين |
| يريد | يبي/يريد | يريد أن يأكل → يبي ياكل |
| يذهب | يروح | يذهب إلى السوق → يروح للسوق |
| يأتي | يجي | يأتي غداً → يجي بكرة |

**مثال تطبيقي**:
```
الفصحى: "أريد أن أذهب إلى السوق لشراء بعض الخضروات"
الخليجية: "أبي أروح للسوق لأشتري شوي خضار"
```

### ٢. التحويل من الفصحى إلى المصرية

**القواعد الأساسية**:

| الفصحى | المصرية | المثال |
|--------|---------|--------|
| ق | أ | قال → أل، قلب → ألب |
| ج | ج (تفخيم) | جاء → جا |
| ك | ك | كيف → إزيك |
| هذا | ده | هذا كتاب → ده كتاب |
| ماذا | إيه/إيه ده | ماذا تريد → عايز إيه |
| كيف | إزيك | كيف حالك → إزيك |
| أنا | أنا | أنا ذاهب → أنا رايح |
| نحن | إحنا | نحن ذاهبون → إحنا رايبين |
| يريد | عايز | يريد أن يأكل → عايز ياكل |
| يذهب | يروح | يذهب إلى السوق → يروح للسوق |
| يأتي | يجي | يأتي غداً → يجي بكرة |
| ليس | مش | ليس جيد → مش كويس |
| جداً | أوي | جيد جداً → كويس أوي |

**مثال تطبيقي**:
```
الفصحى: "هذا الطعام لذيذ جداً، شكراً لك"
المصرية: "الأكل ده لذيذ أوي، شكراً ليك"
```

### ٣. التحويل من الفصحى إلى الشامية (سوري/لبناني/أردني/فلسطيني)

**القواعد الأساسية**:

| الفصحى | الشامية | المثال |
|--------|---------|--------|
| ق | ء/ق | قال → أل، قلب → قلب |
| ك | ك | كيف | كيفك |
| هذا | هاد | هذا كتاب → هاد كتاب |
| ماذا | شو | ماذا تريد → شو بدك |
| كيف | كيف | كيف حالك → كيفك |
| أنا | أنا | أنا ذاهب → أنا رايح |
| نحن | إحنا | نحن ذاهبون → إحنا رايبين |
| يريد | بدو/بيدو | يريد أن يأكل → بدو ياكل |
| يذهب | يروح | يذهب إلى السوق → يروح عالسوق |
| يأتي | بيجي | يأتي غداً → بيجي بكرة |
| ليس | مش | ليس جيد → مش منيح |
| جداً | كتير | جيد جداً → منيح كتير |

**مثال تطبيقي**:
```
الفصحى: "ماذا تريد أن تفعل غداً؟"
الشامية: "شو بدك تعمل بكرة؟"
```

### ٤. التحويل بين اللهجات

عند التحويل بين اللهجات (مثلاً: مصري إلى خليجي):

١. حدد الكلمات المميزة للهجة المصدر
٢. استبدلها بمكافئاتها في اللهجة الهدف
٣. راعِ الفروق النحوية والصرفية
٤. حافظ على المعنى الأصلي

**مثال**:
```
المصرية: "إزك يا باشا، عامل إيه؟"
الخليجية: "شلونك يا غالي، كيف حالك؟"
```

## قواعد اللغة (Language Rules)

### ملاحظات عامة

١. **الحفاظ على الاحترام**: حتى في العامية، استخدم لغة محترمة في السياقات الرسمية

٢. **تجنب الكلمات الغريبة**: استخدم كلمات مفهومة في اللهجة المستهدفة

٣. **مراعاة السياق**: بعض الكلمات قد تكون مقبولة في لهجة وغير لائقة في أخرى

٤. **الأفعال**: انتبه لتصريف الأفعال في كل لهجة

٥. **الضمائر**: تختلف الضمائر بين اللهجات

### جدول الضمائر

| الفصحى | الخليجية | المصرية | الشامية |
|--------|----------|---------|---------|
| أنا | أنا | أنا | أنا |
| أنتَ | إنتَ | إنتَ | إنتَ |
| أنتِ | إنتِ | إنتِ | إنتِ |
| هو | إيوَ/هوَ | هوَ | هوَ |
| هي | هيَ | هيَ | هيَ |
| نحن | إحنا | إحنا | إحنا |
| أنتم | إنتو | إنتوا | إنتو |
| هم | همَ | همَ | همَ |

## مثال (Example)

**الطلب:**
```
حول النص التالي إلى اللهجة الخليجية:
"أهلاً وسهلاً بكم في موقعنا الإلكتروني. نحن نسعى لتقديم أفضل الخدمات لعملائنا الكرام."
```

**النتيجة المتوقعة:**
```
اللهجة الخليجية:
"هلا والله بكم في موقعنا. إحنا نسعى نقدم أفضل الخدمات لعملائنا الكرام."

أو أكثر عامية:
"هلا وغلا بكم بموقعنا. إحنا نبي نقدم أحسن خدمات لعملائنا الأعزاء."
```
