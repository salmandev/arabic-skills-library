## الهدف

إتقان أساسيات الكتابة الإبداعية باللغة العربية.

## أنواع الكتابة الإبداعية

### ١. القصة القصيرة

**العناصر:**
- شخصية رئيسية
- صراع
- ذروة
- حل

### ٢. الرواية

**الهيكل:**
- مقدمة (٢٥٪)
- مواجهة (٥٠٪)
- حل (٢٥٪)

### ٣. الشعر

**الأنواع:**
- شعر عمودي
- شعر حر
- نثر شعري
- شعر تفعيلة

### ٤. السيناريو

**العناصر:**
- حوار
- وصف مشاهد
- إخراج

## تقنيات الكتابة

### ١. الاستعارة

```
"الوقت مال" ← استعارة
```

### ٢. التشبيه

```
"وجهها كالقمر" ← تشبيه
```

### ٣. الشخصنة

```
"الريح تعوي" ← شخصنة
```
