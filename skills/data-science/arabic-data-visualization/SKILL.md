## الهدف

إتقان أساسيات تصور البيانات وإنشاء رسوم بيانية فعالة.

## مبادئ التصور

### ١. اختيار الرسم المناسب

| النوع | الاستخدام |
|-------|----------|
| **Bar Chart** | مقارنة فئات |
| **Line Chart** | اتجاهات زمنية |
| **Scatter Plot** | علاقات متغيرات |
| **Pie Chart** | نسب مئوية |
| **Heatmap** | كثافة بيانات |

### ٢. تصميم فعال

```
✓ بساطة ووضوح
✓ ألوان مناسبة
✓ تسميات واضحة
✓ عنوان وصفي
```

### ٣. تجنب أخطاء شائعة

```
✗ ازدحام معلومات
✗ ألوان مشتتة
✗ محاور مضللة
✗ عدم سياق
```
