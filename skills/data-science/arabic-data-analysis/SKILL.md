## الهدف

إتقان أساسيات تحليل البيانات باستخدام Python و R.

## مراحل التحليل

### ١. جمع البيانات

```
✓ تحديد مصادر
✓ استخراج بيانات
✓ تنظيف بيانات
✓ دمج مجموعات
```

### ٢. استكشاف البيانات

**الإحصاء الوصفي:**
- متوسط، وسيط، منوال
- تباين، انحراف معياري
- ارتباط، تغاير

**التصور:**
- histograms
- scatter plots
- box plots
- heatmaps

### ٣. النمذجة

| النوع | الأمثلة |
|-------|---------|
| **انحدار** | Linear, Logistic |
| **تصنيف** | Decision Tree, SVM |
| **تجميع** | K-Means, Hierarchical |
