## الهدف

إتقان أساسيات معالجة اللغة الطبيعية (NLP) خاصة للعربية.

## مراحل معالجة النص

### ١. المعالجة المسبقة

```
✓ تطبيع نص
✓ إزالة تشكيل
✓ تجزئة (Tokenization)
✓ إزالة stopwords
✓ تجذيع (Stemming/Lemmatization)
```

### ٢. التمثيل

| الطريقة | الوصف |
|---------|-------|
| **Bag of Words** | تكرار كلمات |
| **TF-IDF** | أهمية نسبية |
| **Word2Vec** | تضمين كلمات |
| **BERT** | تضمين سياقي |

### ٣. نماذج شائعة

**تصنيف نص:**
- تحليل مشاعر
- تصنيف مواضيع
- كشف لغة

**توليد نص:**
- ترجمة آلية
- تلخيص تلقائي
- إجابة أسئلة
