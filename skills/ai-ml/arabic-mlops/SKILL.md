## الهدف

إتقان أساسيات MLOps (DevOps للتعلم الآلي).

## مكونات MLOps

### ١. دورة حياة النموذج

```
١. جمع بيانات
٢. تدريب نموذج
٣. تقييم
٤. نشر
٥. مراقبة
٦. إعادة تدريب
```

### ٢. إدارة نماذج

| الأداة | الوصف |
|--------|-------|
| **MLflow** | تتبع تجارب |
| **DVC** | إصدار بيانات |
| **Weights & Biases** | تتبع وتصور |

### ٣. نشر نماذج

**الطرق:**
- REST API
- Batch Inference
- Edge Deployment
- Serverless

### ٤. مراقبة

**المقاييس:**
- دقة تنبؤات
- انحراف بيانات
- زمن استجابة
- استخدام موارد
