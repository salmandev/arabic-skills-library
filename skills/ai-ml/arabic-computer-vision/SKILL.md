## الهدف

إتقان أساسيات الرؤية الحاسوبية (Computer Vision) ومعالجة الصور.

## مفاهيم أساسية

### ١. معالجة الصور

**العمليات:**
- تغيير حجم (Resizing)
- تحويل رمادي (Grayscale)
- تصفية (Filtering)
- كشف حواف (Edge Detection)

### ٢. شبكات CNN

**الطبقات:**
- Convolutional Layers
- Pooling Layers
- Fully Connected Layers

### ٣. تطبيقات شائعة

| التطبيق | الوصف |
|---------|-------|
| **تصنيف صور** | تحديد محتوى الصورة |
| **كشف أشياء** | تحديد مواقع الأشياء |
| **تجزئة صور** | تقسيم الصورة لمناطق |
| **تعرف وجوه** | التعرف على الوجوه |

## مكتبات شائعة

| المكتبة | الوصف |
|---------|-------|
| **OpenCV** | معالجة صور وفيديو |
| **PIL/Pillow** | معالجة صور بسيطة |
| **torchvision** | رؤية مع PyTorch |
| **tf.image** | رؤية مع TensorFlow |

## أفضل الممارسات

✅ زيادة بيانات (Data Augmentation)
✅ شبكات Pre-trained
✅ تقييم شامل
✅ معالجة مسبقة صحيحة

❌ لا تهمل حجم البيانات
❌ لا تستخدم صور منخفضة الجودة
❌ لا تهمل overfitting
