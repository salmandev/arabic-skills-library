## الهدف

إتقان أساسيات تطوير الواجهات الأمامية (Frontend Development).

## تقنيات أساسية

### ١. HTML

**الهيكل:**
```html
<!DOCTYPE html>
<html>
<head>
    <title>صفحتي</title>
</head>
<body>
    <h1>مرحباً</h1>
</body>
</html>
```

### ٢. CSS

**التنسيق:**
```css
body {
    font-family: Arial;
    background: #f0f0f0;
}

h1 {
    color: #333;
}
```

### ٣. JavaScript

**التفاعل:**
```javascript
document.querySelector('h1').addEventListener('click', () => {
    alert('مرحباً!');
});
```
