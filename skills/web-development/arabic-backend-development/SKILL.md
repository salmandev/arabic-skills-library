## APIs

**REST:**
```
GET /users - جلب مستخدمين
POST /users - إنشاء مستخدم
PUT /users/1 - تحديث مستخدم
DELETE /users/1 - حذف مستخدم
```

**GraphQL:**
```graphql
query {
  user(id: 1) {
    name
    email
  }
}
```
