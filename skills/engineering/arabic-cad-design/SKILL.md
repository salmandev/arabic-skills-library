## الهدف

إتقان أساسيات التصميم بمساعدة الحاسوب (CAD) للرسم الهندسي.

## برامج CAD شائعة

| البرنامج | الاستخدام | الشركة |
|----------|----------|--------|
| **AutoCAD** | رسم ثنائي وثلاثي الأبعاد | Autodesk |
| **SolidWorks** | تصميم ميكانيكي | Dassault |
| **Revit** | نمذجة معلومات بناء | Autodesk |
| **Fusion 360** | تصميم وتصنيع | Autodesk |
| **CATIA** | تصميم متقدم | Dassault |

## أساسيات الرسم

### أوامر أساسية

```
LINE      - رسم خطوط
CIRCLE    - رسم دوائر
RECTANGLE - رسم مستطيلات
TRIM      - قص أجزاء
OFFSET    - إنشاء نسخ متوازية
MIRROR    - عكس كائنات
ARRAY     - تكرار بنمط
ROTATE    - تدوير
SCALE     - تغيير مقياس
DIMENSION - إضافة أبعاد
```

### طبقات الرسم (Layers)

```
Layer 00-Walls    - الجدران
Layer 00-Doors    - الأبواب
Layer 00-Windows  - النوافذ
Layer 00-Text     - النصوص
Layer 00-Dims     - الأبعاد
Layer 00-Electrical - الكهرباء
Layer 00-Plumbing - السباكة
```
