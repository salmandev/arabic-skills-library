## الهدف

إتقان أساسيات مراقبة الجودة في الهندسة والتصنيع.

## أدوات الجودة

### ٧ أدوات أساسية

1. **مخطط السبب والأثر** (Ishikawa)
2. **مخطط باريتو** (Pareto)
3. **المدرج التكراري** (Histogram)
4. **ورقة الفحص** (Check Sheet)
5. **مخطط التشتت** (Scatter Diagram)
6. **خريطة التحكم** (Control Chart)
7. **مخطط الانسياب** (Flow Chart)

## مبادئ الجودة

### PDCA Cycle

```
Plan (خطّط) → Do (نفّذ) → Check (تحقّق) → Act (تصرّف)
```

### Six Sigma

```
DMAIC: Define → Measure → Analyze → Improve → Control
```
