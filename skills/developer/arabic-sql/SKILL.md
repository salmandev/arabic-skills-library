## الهدف

تعلم استعلامات SQL بالعربية.

## أساسيات

```sql
-- تحديد بيانات
SELECT name, email FROM users WHERE age > 25;

-- إضافة
INSERT INTO users (name, email) VALUES ('أحمد', 'ahmed@example.com');

-- تحديث
UPDATE users SET age = 26 WHERE name = 'أحمد';

-- حذف
DELETE FROM users WHERE age < 18;
```
