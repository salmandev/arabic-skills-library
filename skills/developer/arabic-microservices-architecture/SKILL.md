## الهدف

إتقان أساسيات عمارة Microservices وتصميم أنظمة موزعة.

## مبادئ Microservices

### ١. خدمات مستقلة

**الخصائص:**
- خدمة واحدة = مسؤولية واحدة
- نشر مستقل
- قواعد بيانات منفصلة
- اتصال عبر APIs

### ٢. اتصال خدمات

**الأنماط:**
- متزامن (HTTP/REST, gRPC)
- غير متزامن (Message Queues)
- Event-driven

### ٣. اكتشاف خدمات

**الأدوات:**
- Service Registry
- Service Discovery
- API Gateway

### ٤. مرونة

**الأنماط:**
- Circuit Breaker
- Retry Pattern
- Bulkhead Pattern
- Fallback Pattern
