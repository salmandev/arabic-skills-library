## الهدف

تعلم أساسيات سطر أوامر لينكس بالعربية.

## أوامر أساسية

```bash
# تنقل
pwd          # مجلد الحالي
ls           # عرض ملفات
cd /path     # تغيير مجلد

# ملفات
touch file   # إنشاء ملف
cat file     # عرض محتوى
rm file      # حذف ملف

# صلاحيات
chmod 755 file
chown user:group file
```
