## الهدف

إتقان أساسيات التسويق الرقمي.

## قنوات التسويق

### ١. محركات بحث (SEM)

**SEO:**
- تحسين محتوى
- بناء روابط
- تحسين تقني

**PPC:**
- إعلانات Google
- إعلانات Bing
- دفع لكل نقرة

### ٢. وسائل تواصل

| المنصة | الجمهور | النوع |
|--------|--------|-------|
| **Facebook** | عام | صور، فيديو |
| **Instagram** | شباب | صور، Stories |
| **Twitter** | محترفون | نص، أخبار |
| **LinkedIn** | أعمال | مهني، B2B |
| **TikTok** | شباب | فيديو قصير |

### ٣. بريد إلكتروني

```
✓ قائمة بريدية
✓ تقسيم جمهور
✓ أتمتة حملات
✓ تحليل نتائج
```

## مقاييس أداء

| المقياس | الوصف |
|--------|-------|
| **CTR** | نسبة نقرات |
| **CPC** | تكلفة نقرة |
| **CPA** | تكلفة اكتساب |
| **ROI** | عائد استثمار |
| **Conversion Rate** | نسبة تحويل |
