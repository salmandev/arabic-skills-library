## الهدف

إتقان أساسيات أمن الشبكات وحمايتها.

## طبقات الحماية

### ١. جدار الحماية (Firewall)

**الأنواع:**
- Packet Filtering
- Stateful Inspection
- Next-Gen Firewall

### ٢. كشف ومنع التسلل (IDS/IPS)

| النوع | الوظيفة |
|-------|---------|
| **IDS** | كشف فقط |
| **IPS** | كشف ومنع |

### ٣. شبكة خاصة افتراضية (VPN)

**الاستخدام:**
- وصول آمن عن بُعد
- تشفير البيانات
- إخفاء الهوية

### ٤. تجزئة الشبكة

```
شبكة إدارة ← معزولة
شبكة إنتاج ← معزولة
شبكة ضيوف ← معزولة
```

## أفضل الممارسات

✅ جدران حماية متعددة الطبقات
✅ تحديث مستمر
✅ مراقبة حركة
✅ سجلات أمنية

❌ لا تفتح منافذ غير ضرورية
❌ لا تهمل التحديثات
❌ لا تستخدم كلمات مرور ضعيفة
