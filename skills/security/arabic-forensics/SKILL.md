## الهدف

إتقان أساسيات التحقيقات الجنائية الرقمية وجمع الأدلة.

## مراحل التحقيق

### ١. التحديد

```
□ تحديد مصادر الأدلة
□ تحديد نطاق التحقيق
□ الحصول على الأذونات
```

### ٢. الحفظ

```
□ عزل الأجهزة
□ إنشاء صورة طبق الأصل
□ توثيق السلسلة
```

### ٣. التحليل

```
□ استعادة ملفات محذوفة
□ تحليل سجلات
□ استخراج بيانات
```

### ٤. الإبلاغ

```
□ توثيق النتائج
□ إعداد تقرير خبير
□ الشهادة في المحكمة
```

## أدوات شائعة

| الأداة | الاستخدام |

❌ لا تكسر سلسلة الحيازة
