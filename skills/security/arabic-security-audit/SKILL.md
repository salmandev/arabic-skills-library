## الهدف

إجراء تدقيقات أمنية شاملة باللغة العربية.

## مراحل التدقيق

### ١. التخطيط

```
□ تحديد النطاق
□ تحديد المعايير (ISO 27001, NCA ECC)
□ تشكيل فريق التدقيق
□ جدولة المقابلات
□ إعداد قوائم الفحص
```

### ٢. التنفيذ

**مجالات التدقيق:**

| المجال | النقاط الرئيسية |
|--------|----------------|
| **السياسات** | وجود، تحديث، تطبيق |
| **الوصول** | IAM، صلاحيات، MFA |
| **الشبكة** | جدران حماية، تجزئة |
| **الأجهزة** | تحديثات، حماية نقاط نهاية |
| **البيانات** | تشفير، نسخ احتياطي |
| **الوعي** | تدريب، اختبارات تصيد |

### ٣. التقرير

**هيكل التقرير:**
```
١. ملخص تنفيذي
٢. منهجية التدقيق
٣. النتائج
٤. التوصيات
٥. خطة العلاج
```

## قائمة فحص مختصرة

```
□ سياسات أمنية موثقة
□ جرد أصول محدث
□ تقييم مخاطر سنوي
□ MFA مفعل
□ تحديثات أمنية
□ نسخ احتياطي مختبر
□ خطة استجابة حوادث
□ تدريب وعي أمني
□ اختبارات اختراق
□ مراجعة صلاحيات
```

## أفضل الممارسات

✅ وثّق كل شيء
✅ قدم أدلة
✅ رتب حسب الأولوية
✅ قدم توصيات عملية
✅ تابع العلاج

❌ لا تهمل السياق
❌ لا تكن عاماً جداً
❌ لا تهمل المتابعة
❌ لا تنسَ الإيجابيات
