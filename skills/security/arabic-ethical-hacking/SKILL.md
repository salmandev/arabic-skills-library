## الهدف

إتقان أساسيات الاختراق الأخلاقي واختبار الاختراق.

## مراحل الاختراق

### ١. الاستطلاع

```
□ جمع معلومات
□ فحص شبكي
□ تحديد أهداف
```

### ٢. المسح

```
□ فحص منافذ
□ تحديد خدمات
□ كشف ثغرات
```

### ٣. الاستغلال

```
□ استغلال ثغرات
□ الحصول على وصول
□ الحفاظ على الوصول
```

### ٤. الإبلاغ

```
□ توثيق النتائج
□ تقييم المخاطر
□ توصيات علاج
```

## أفضل الممارسات

✅ احصل على إذن كتابي
✅ وثّق كل شيء
✅ التزم بالنطاق
✅ قدّم توصيات عملية

❌ لا تختبر بدون إذن
❌ لا تتجاوز النطاق
❌ لا تكشف معلومات حساسة
