## الهدف

إنشاء وإدارة حملات تسويقية فعالة عبر البريد الإلكتروني باللغة العربية.

## أنواع الإيميلات

### ١. إيميلات ترحيبية

**المحتوى:**
```
الموضوع: أهلاً بك في [الاسم]! 👋

مرحباً [الاسم]،

شكراً لانضمامك إلينا!

ما يمكنك توقعه:
✓ [فائدة ١]
✓ [فائدة ٢]
✓ [فائدة ٣]

ابدأ الآن: [CTA]

فريق [الاسم]
```

### ٢. نشرات إخبارية

**التردد:** أسبوعي أو شهري

**المحتوى:**
- ملخص الأسبوع
- مقالات جديدة
- عروض خاصة
- أخبار الشركة

### ٣. إيميلات ترويجية

**المناسبات:**
- تخفيضات موسمية
- إطلاق منتج جديد
- عروض محدودة
- عيد ميلاد العميل

### ٤. إيميلات تعليمية

**الأنواع:**
- دورة عبر البريد
- نصائح أسبوعية
- أدلة شاملة
- دراسات حالة

## كتابة موضوع جذاب

### صيغ مجربة

```
✅ "٥ طرق لـ..."
✅ "كيف تحصل على..."
✅ "السر وراء..."
✅ "عرض خاص لك..."
✅ "آخر فرصة..."
✅ "تهانينا!"
✅ "دعوتك الحصرية..."
```

### أفضل الممارسات

```
✓ اجعل الموضوع قصير (<٥٠ حرف)
✓ استخدم أرقام
✓ أضف طابع شخصي
✓ خلق فضول
✓ تجنب كلمات سبام
```

## مقاييس النجاح

| المقياس | الصيغة | المتوسط |
|---------|--------|---------|
| **معدل الفتح** | (إيميلات مفتوحة ÷ مرسلة) × ١٠٠ | ٢٠٪ |
| **معدل النقر** | (نقرات ÷ إيميلات مفتوحة) × ١٠٠ | ٢.٥٪ |
| **معدل التحويل** | (تحويلات ÷ نقرات) × ١٠٠ | ٥٪ |
| **معدل الارتداد** | (إيميلات مرتدة ÷ مرسلة) × ١٠٠ | <٢٪ |
| **معدل إلغاء الاشتراك** | (إلغاءات ÷ مرسلة) × ١٠٠ | <٠.٥٪ |

## تقسيم القائمة

### طرق التقسيم

```
📊 ديموغرافي: عمر، جنس، موقع
📊 سلوكي: شراء سابق، تفاعل
📊 مرحلة: عميل جديد، محتمل، نشط
📊 اهتمامات: فئات منتجات
```

## أتمتة الإيميلات

### سيناريوهات شائعة

```
١. سلسلة ترحيبية (٥ إيميلات)
   - يوم ١: ترحيب
   - يوم ٣: قصة الشركة
   - يوم ٥: عرض خاص
   - يوم ٧: شهادة عميل
   - يوم ١٠: CTA نهائي

٢. عربة متروكة (٣ إيميلات)
   - بعد ساعة: تذكير
   - بعد يوم: عرض مساعدة
   - بعد ٣ أيام: خصم ١٠٪

٣. إعادة تفاعل (٣ إيميلات)
   - بعد ٣٠ يوم خمول: "اشتقنا لك"
   - بعد ٦٠ يوم: عرض خاص
   - بعد ٩٠ يوم: آخر محاولة
```

## أفضل الممارسات

✅ استخدم اسم حقيقي في المرسل
✅ أرسل في أوقات مناسبة (١٠ ص - ٢ م)
✅ اختبر A/B باستمرار
✅ نظف القائمة من غير النشطين
✅ اجعل الإيميل متجاوب
✅ تتبع المقاييس

❌ لا ترسل من no-reply@
❌ لا ترسل في عطلات
❌ لا تهمل إلغاء الاشتراك
❌ لا ترسل بدون اختبار
