## الهدف

إجراء مقابلات عمل احترافية وفعالة باللغة العربية لتقييم المرشحين بدقة.

## أنواع المقابلات

| النوع | الوصف | الاستخدام |
|-------|-------|----------|
| **مقابلة هاتفية** | فحص أولي | تصفية مرشحين |
| **مقابلة فردية** | وجهاً لوجه | تقييم متعمق |
| **مقابلة لجنة** | عدة محاورين | مناصب عليا |
| **مقابلة سلوكية** | أسئلة عن مواقف | تقييم كفاءات |
| **مقابلة تقنية** | أسئلة تخصصية | وظائف تقنية |

## مراحل المقابلة

### ١. التحضير (١٥ دقيقة)

```
□ راجع السيرة الذاتية
□ حدد الكفاءات المطلوبة
□ جهز الأسئلة
□ احجز غرفة مناسبة
□ اختبر التقنية (إن وجدت)
```

### ٢. الافتتاح (٥ دقائق)

```
١. رحّب بالمرشح بحرارة
٢. قدّم نفسك والمنصب
٣. اشرح هيكل المقابلة
٤. كسر الجليد بأسئلة بسيطة
```

**مثال:**
```
"أهلاً وسهلاً بك يا أحمد. أنا محمد من إدارة الموارد البشرية.
شكراً لحضورك اليوم. المقابلة ستستغرق حوالي ٤٥ دقيقة،
وسأتحدث فيها عن خبرتك ومهاراتك. هل لديك أي أسئلة قبل البدء؟"
```

### ٣. الأسئلة الأساسية (٣٠ دقيقة)

#### أسئلة تعريفية

```
١. "حدثنا عن نفسك باختصار"
٢. "ما الذي تعرفه عن شركتنا؟"
٣. "لماذا تقدمت لهذا المنصب؟"
```

#### أسئلة سلوكية (STAR Method)

```
١. "أخبرني عن موقف واجهت فيه تحدياً صعباً في العمل. كيف تعاملت معه؟"
   - الموقف (Situation)
   - المهمة (Task)
   - الإجراء (Action)
   - النتيجة (Result)

٢. "صف لي مرة عملت فيها ضمن فريق. ما كان دورك؟"

٣. "أخبرني عن مرة ارتكبت فيها خطأً. كيف تعاملت معه؟"
```

#### أسئلة الكفاءة

```
للقيادة:
"كيف تحفز فريقك عندما يكون معنوياتهم منخفضة؟"

لحل المشكلات:
"صف لي عملية اتخاذ قرار صعب اتخذته مؤخراً"

للتواصل:
"كيف تتعامل مع زميل صعب المراس؟"
```

#### أسئلة تقنية (حسب المنصب)

```
لمحاسب:
"ما هي القوائم المالية الأساسية؟"

لمبرمج:
"ما الفرق بين GET و POST؟"

لبائع:
"كيف تتعامل مع اعتراض عميل على السعر؟"
```

### ٤. أسئلة المرشح (٥ دقائق)

```
"هل لديك أي أسئلة تود طرحها؟"
```

**أسئلة متوقعة من المرشح:**
- "ما هي ثقافة الشركة؟"
- "ما هي فرص التطوير؟"
- "ما هي الخطوات التالية؟"

### ٥. الختام (٥ دقائق)

```
١. اشكر المرشح على وقته
٢. اشرح الخطوات التالية
٣. حدد موعد الإشعار
٤. ودّع بطريقة احترافية
```
