## الهدف

تقديم الإقرارات الضريبية باللغة العربية وفقاً لمتطلبات هيئة الزكاة والضريبة والجمارك (ZATCA).

## أنواع الضرائب

### ١. ضريبة القيمة المضافة (VAT)

**المعدل**: ١٥٪  
**التسجيل**: إلزامي إذا تجاوز ٣٧٥,٠٠٠ ريال سنوياً

**الإقرار الدوري:**
```
الإيرادات الخاضعة للضريبة: _______________
ضريبة المبيعات (١٥٪): _______________
ضريبة المشتريات: _______________
صافي الضريبة المستحقة: _______________
```

### ٢. ضريبة الدخل

**للأفراد**: ٠٪ (للدخل الشخصي)  
**للشركات الأجنبية**: ٢٠٪

### ٣. الزكاة

**المعدل**: ٢.٥٪ من الوعاء الزكوي  
**للشركات السعودية والخليجية**

## خطوات التقديم

### ١. التسجيل في بوابة ZATCA

```
١. ادخل على portal.zatca.gov.sa
٢. سجل دخول أو أنشئ حساب
٣. اكمل بيانات المنشأة
٤. احصل على شهادة التسجيل
```

### ٢. إعداد الفاتورة الإلكترونية

```
البيانات الإلزامية:
□ اسم المنشأة
□ الرقم الضريبي (١٥ رقم)
□ العنوان
□ تاريخ الفاتورة
□ رقم الفاتورة
□ وصف السلع/الخدمات
□ المبلغ قبل الضريبة
□ الضريبة (١٥٪)
□ الإجمالي
□ QR Code
```

### ٣. تقديم الإقرار

```
١. سجل الدخول للبواية
٢. اختر "الإقرارات الضريبية"
٣. اختر الفترة الضريبية
٤. أدخل البيانات:
   - المبيعات
   - المشتريات
   - الضريبة المحصلة
   - الضريبة المدفوعة
٥. راجع البيانات
٦. قدم الإقرار
٧. سدد المبلغ المستحق
```
