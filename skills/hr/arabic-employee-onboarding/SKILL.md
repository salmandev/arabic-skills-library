### ١. قبل وصول

```
□ إرسال عرض عمل
□ تجهيز معدات
□ إنشاء حسابات
□ جدول أول أسبوع
```

### ٢. يوم أول

```
✓ ترحيب
✓ جولة مكتب
✓ لقاء فريق
✓ تدريب أنظمة
```

### ٣. أول أسبوع

```
✓ أهداف واضحة
✓ تدريب دورات
✓ لقاءات زملاء
✓ تغذية راجعة
```

```
✓ أهداف ٣٠-٦٠-٩٠
```
