## الهدف

إتقان أساسيات التدريب وتطوير الموظفين.

## دورة التدريب

### ١. تحليل احتياج

```
□ تقييم أداء
□ تحديد فجوات
□ أولويات تدريب
□ ميزانية
```

### ٢. تصميم برنامج

```
□ أهداف تدريبية
□ محتوى
□ طريقة تقديم
□ مدة
```

### ٣. تنفيذ

```
✓ مدرب داخلي/خارجي
✓ قاعة/أونلاين
✓ مواد تدريبية
✓ أنشطة
```

### ٤. تقييم

**مستويات التقييم:**
- رد فعل متدربين
- تعلم
- سلوك
- نتائج
