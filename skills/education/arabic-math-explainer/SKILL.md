## مثال (Example)

**الطلب:**
```
اشرح مفهوم المعادلة التربيعية مع الأمثلة.
```

**النتيجة المتوقعة:**
```
# المعادلة التربيعية

## التعريف

المعادلة التربيعية هي معادلة جبرية من الدرجة الثانية على الصورة:

أس² + ب س + جـ = ٠

حيث:
- أس²: الحد التربيعي
- ب س: الحد الخطي
- جـ: الحد الثابت
- أ، ب، جـ: أعداد حقيقية، أ ≠ ٠

## طرق الحل

### ١. التحليل إلى عوامل

**المثال**: حل المعادلة: س² + ٥س + ٦ = ٠

**الحل**:
١. نبحث عن عددين حاصل ضربهما = ٦ ومجموعهما = ٥
٢. العددان هما: ٢، ٣
٣. نحلل: (س + ٢)(س + ٣) = ٠
٤. إذن: س = -٢ أو س = -٣

### ٢. القانون العام

القانون: س = (-ب ± √(ب² - ٤أجـ)) / ٢أ

**المثال**: حل المعادلة: ٢س² + ٧س + ٣ = ٠

**الحل**:
أ = ٢، ب = ٧، جـ = ٣

١. نحسب المميز: ب² - ٤أجـ = ٤٩ - ٢٤ = ٢٥
٢. س = (-٧ ± √٢٥) / ٤
٣. س = (-٧ ± ٥) / ٤
٤. الحلان: س = -١/٢ أو س = -٣

## تمارين للتطبيق

١. حل بالمعادلة: س² - ٩ = ٠
٢. حل بالتحليل: س² - ٧س + ١٢ = ٠
٣. حل بالقانون: ٣س² - ٥س + ٢ = ٠

**الإجابات**:
١. س = ٣ أو س = -٣
٢. س = ٣ أو س = ٤
٣. س = ١ أو س = ٢/٣
```
