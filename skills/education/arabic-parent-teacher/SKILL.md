## الهدف

إجراء اجتماعات فعالة مع أولياء الأمور باللغة العربية لدعم تعلم الطلاب.

## التحضير للاجتماع

### قبل الاجتماع

```
□ رتب جدول المواعيد
□ جهز أعمال الطالب
□ جمع بيانات التقييم
□ حدد نقاط القوة والضعف
□ جهز توصيات
□ رتب مكان الاجتماع
```

### مواد الاجتماع

```
□ عينات من عمل الطالب
□ نتائج الاختبارات
□ سجل الحضور
□ ملاحظات سلوكية
□ نموذج تقرير الاجتماع
```

## هيكل الاجتماع

### ١. الافتتاح (٥ دقائق)

```
"أهلاً وسهلاً بكم. شكراً لحضوركم.
الهدف من اجتماعنا اليوم هو مناقشة تقدم [اسم الطالب].
سيستغرق الاجتماع حوالي ٢٠ دقيقة."
```

### ٢. نقاط القوة (٥ دقائق)

```
"دعني أبدأ بالإيجابيات:
• [اسم الطالب] متميز في...
• ألاحظ تحسناً في...
• يشارك بفعالية في..."
```

### ٣. مجالات التحسين (٥ دقائق)

```
"هناك بعض المجالات التي تحتاج دعم:
• يحتاج تحسين في...
• ألاحظ صعوبة في...
• أقترح..."
```

### ٤. خطة العمل (٥ دقائق)

```
"خطة الدعم المقترحة:
• في المدرسة: _______________
• في المنزل: _______________
• المتابعة: _______________"
```

### ٥. الختام (٥ دقائق)

```
"شكراً لوقتكم. هل لديكم أسئلة؟
يمكنكم التواصل معي عبر...
موعد المتابعة: _______________"
```

## التعامل مع المواقف الصعبة

### ولي أمر غاضب

```
✅ استمع بهدوء
✅ اعترف بمشاعره
✅ قدم حقائق
✅ اقترح حلولاً
✅ وثّق المحادثة
```

### ولي أمر غير مهتم

```
✅ أكد على أهمية المشاركة
✅ اقترح طرق سهلة للمساعدة
✅ أرسل تحديثات منتظمة
✅ قدّر أي مشاركة
```

## أفضل الممارسات

✅ ابدأ بالإيجابيات
✅ كن محدداً بأمثلة
✅ استمع بفعالية
✅ تجنب المصطلحات التربوية
✅ اتفق على خطة عمل
✅ تابع بعد الاجتماع

❌ لا تنتقد الطالب شخصياً
❌ لا تعد بوعود لا تستطيع الوفاء بها
❌ لا تقارن الطالب بزملائه
❌ لا تهمل ملاحظات ولي الأمر
