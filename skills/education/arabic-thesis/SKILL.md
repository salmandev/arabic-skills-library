## الهدف

إرشاد الطلاب والباحثين في كتابة الأطروحات والأبحاث العلمية باللغة العربية.

## هيكل الأطروحة

### ١. الصفحات الأولية

```
□ صفحة العنوان
□ صفحة التوقيع
□ الإهداء
□ الشكر والتقدير
□ الملخص (عربي)
□ الملخص (إنجليزي)
□ جدول المحتويات
□ قائمة الجداول
□ قائمة الأشكال
□ قائمة الاختصارات
```

### ٢. الفصول الرئيسية

```
الفصل ١: المقدمة
├── خلفية البحث
├── مشكلة البحث
├── أسئلة البحث
├── فرضيات البحث
├── أهداف البحث
├── أهمية البحث
├── حدود البحث
└── تعريف المصطلحات

الفصل ٢: الدراسات السابقة
├── مقدمة
├── الدراسات العربية
├── الدراسات الأجنبية
├── التعليق على الدراسات
└── خلاصة

الفصل ٣: المنهجية
├── منهج البحث
├── مجتمع البحث
├── عينة البحث
├── أدوات البحث
├── إجراءات البحث
└── المعالجة الإحصائية

الفصل ٤: النتائج
├── عرض النتائج
├── جداول وأشكال
├── تحليل النتائج
└── مناقشة النتائج

الفصل ٥: التوصيات
├── خلاصة البحث
├── التوصيات
├── المقترحات
└── المراجع
```

### ٣. المراجع

**أنماط شائعة:**

| النمط | الاستخدام |
|-------|----------|
| **APA** | علوم اجتماعية، تربوية |
| **MLA** | علوم إنسانية، لغات |
| **Chicago** | تاريخ، علوم سياسية |
| **IEEE** | هندسة، حاسوب |
| **Vancouver** | طب، علوم صحية |

**مثال APA:**
```
الكتاب:
الأخير، الأول. (السنة). عنوان الكتاب. دار النشر.

المقالة:
الأخير، الأول. (السنة). عنوان المقالة. اسم المجلة، عدد(عدد)، صفحات.

الموقع:
اسم الموقع. (تاريخ النشر). عنوان الصفحة. URL
```

## نصائح الكتابة

### الكتابة الأكاديمية

```
✅ استخدم لغة رسمية
✅ تجنب الضمائر الشخصية
✅ استخدم المصطلحات الدقيقة
✅ وثّق كل معلومة
✅ راجع القواعد والإملاء
```

### الاقتباس

```
✅ اقتبس بدقة
✅ استخدم علامات الاقتباس
✅ اذكر المصدر
✅ تجنب الانتحال
```

### التنسيق

```
✅ اتبع دليل الجامعة
✅ رقم الصفحات
✅ استخدم أنماط العناوين
✅ راجع الهوامش
```

## أفضل الممارسات

✅ ابدأ مبكراً
✅ ناقش مع المشرف بانتظام
✅ احتفظ بنسخ احتياطية
✅ استخدم برامج إدارة المراجع
✅ راجع بدقة قبل التسليم

❌ لا تؤجل الكتابة
❌ لا تنسخ من مصادر
❌ لا تهمل التنسيق
❌ لا تنسَ الشكر والتقدير
