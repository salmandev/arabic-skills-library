## الهدف

تطبيق استراتيجيات فعالة لإدارة الفصل الدراسي باللغة العربية.

## استراتيجيات الإدارة

### ١. وضع القواعد

**قواعد فعالة:**
```
✓ كن واضحاً ومحترماً
✓ استمع للآخرين
✓ احترم ممتلكات الآخرين
✓ ارفع يدك قبل الكلام
✓ كن مستعداً للتعلم
```

### ٢. الروتين اليومي

**بداية الحصة:**
```
١. الترحيب بالطلاب
٢. مراجعة الحضور
٣. مراجعة الدرس السابق
٤. عرض أهداف الدرس
٥. بدء النشاط الأول
```

**نهاية الحصة:**
```
١. تلخيص الدرس
٢. الإجابة على أسئلة
٣. تكليف الواجب
٤. تنظيم الفصل
٥. الوداع
```

### ٣. التعامل مع السلوكيات

| السلوك | الاستراتيجية |
|--------|-------------|
| حديث جانبي | اقترب من الطالب، تواصل بصري |
| تأخير | تذكير بالقواعد، عواقب متدرجة |
| عدم تحضير | اتصال ولي الأمر، خطة تحسين |
| عدوانية | فصل فوري، تحويل للإدارة |
