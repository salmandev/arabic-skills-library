## الهدف

إنشاء وتطبيق أدوات تقييم الطلاب باللغة العربية.

## أنواع التقييم

### ١. التقييم التشخيصي

**الغرض:** قياس المعرفة المسبقة

**أدوات:**
- اختبار قبلي
- عصف ذهني
- أسئلة استكشافية

### ٢. التقييم التكويني

**الغرض:** تحسين التعلم أثناء العملية

**أدوات:**
- أسئلة صفية
- واجبات قصيرة
- ملاحظات
-Exit Tickets

### ٣. التقييم الختامي

**الغرض:** قياس التحصيل النهائي

**أدوات:**
- اختبارات نهائية
- مشاريع
- عروض
- ملفات إنجاز

## بناء الاختبارات

### أسئلة اختيار من متعدد

```
السؤال: ما هي عاصمة السعودية؟

أ) جدة
ب) الرياض ✓
ج) مكة
د) الدمام

شروط جيدة:
- سؤال واضح
- إجابة واحدة صحيحة
- مشتتات معقولة
- تجنب "كل ما سبق"
```

### أسئلة الصواب والخطأ

```
ضع علامة (✓) للصواب و (✗) للخطأ:

١. الماء يتكون من ذرتي هيدروجين وذرة أكسجين ( )
٢. الأرض تدور حول الشمس ( )
٣. سرعة الضوء أقل من سرعة الصوت ( )
```

### أسئلة المطابقة

```
طابق بين العمودين:

العمود أ          العمود ب
١. المملكة        أ) القاهرة
٢. مصر           ب) الرياض
٣. الإمارات       ج) باريس
٤. فرنسا         د) أبوظبي
```

### أسئلة الإجابة القصيرة

```
أجب عن الأسئلة التالية:

١. ما هي عاصمة فرنسا؟
الإجابة: _______________

٢. كم عدد قارات العالم؟
الإجابة: _______________
```

### أسئلة المقالية

```
أجب عن السؤال التالي في حدود ٢٠٠ كلمة:

"ناقش أسباب الحرب العالمية الأولى ونتائجها"

معايير التصحيح:
- المحتوى (٤٠ نقطة)
- التنظيم (٢٠ نقطة)
- اللغة (٢٠ نقطة)
- الأمثلة (٢٠ نقطة)
```
