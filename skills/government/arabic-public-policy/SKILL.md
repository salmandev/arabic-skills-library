## الهدف

إتقان أساسيات تطوير وتحليل السياسة العامة.

## دورة السياسة العامة

### ١. تحديد المشكلة

```
□ جمع بيانات
□ تحليل الوضع
□ تحديد الأولويات
```

### ٢. وضع الأجندة

```
□ كسب التأييد
□ حشد الدعم
□ تحديد صانعي القرار
```

### ٣. صياغة السياسة

```
□ بحث وخيارات
□ استشارة أصحاب المصلحة
□ مسودة السياسة
```

### ٤. التنفيذ

```
□ تخصيص موارد
□ إنشاء برامج
□ تدريب موظفين
```

### ٥. التقييم

```
□ جمع بيانات أداء
□ تحليل النتائج
□ توصيات تحسين
```
