## الهدف

إتقان أساسيات إدارة العلاقات مع وسائل الإعلام.

## وظائف العلاقات الإعلامية

### ١. التواصل مع الإعلام

```
□ بناء قائمة إعلامية
□ إرسال بيانات صحفية
□ ترتيب مقابلات
□ تنظيم مؤتمرات
```

### ٢. إدارة السمعة

```
□ مراقبة تغطية
□ الرد على استفسارات
□ إدارة أزمات
□ تعزيز صورة إيجابية
```

### ٣. إنتاج محتوى

```
□ بيانات صحفية
□ مقالات رأي
□ قصص إخبارية
□ محتوى رقمي
```
