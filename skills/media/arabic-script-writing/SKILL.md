## الهدف

إتقان أساسيات كتابة السيناريو للأفلام والمسلسلات.

## هيكل السيناريو

### الهيكل الثلاثي

| الفصل | النسبة | الوصف |
|-------|--------|-------|
| **١** | ٢٥٪ | الإعداد والشخصيات |
| **٢** | ٥٠٪ | المواجهة والصراع |
| **٣** | ٢٥٪ | الذروة والحل |

### نقاط التحول

```
١. الحافز (١٠٪)
٢. نقطة اللاعودة (٢٥٪)
٣. منتصف القصة (٥٠٪)
٤. كل شيء ضاع (٧٥٪)
٥. الذروة (٩٠٪)
```

## تنسيق السيناريو

```
مشهد ١
داخلي. مقهى - نهار

أحمد (٣٠) يجلس وحيداً يشرب قهوة.
يدخل خالد مسرعاً.

                    خالد
          (متنفس)
     أحمد، عندنا مشكلة كبيرة!

أحمد يضع القهوة ببطء.

                    أحمد
     ما نوع المشكلة هذه المرة؟
```
