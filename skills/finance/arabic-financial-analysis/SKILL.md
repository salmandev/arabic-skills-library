## الهدف

إتقان أساسيات التحليل المالي وتقييم الشركات.

## القوائم المالية

### ١. قائمة الدخل

```
الإيرادات
- تكلفة البضاعة
= إجمالي الربح
- مصاريف تشغيل
= الدخل التشغيلي
- ضرائب وفوائد
= صافي الدخل
```

### ٢. الميزانية العمومية

```
الأصول = الخصوم + حقوق الملكية
```

### ٣. قائمة التدفق النقدي

```
تدفق من عمليات
تدفق من استثمار
تدفق من تمويل
= صافي تغيير نقد
```

## نسب مالية

| النسبة | الصيغة | التفسير |
|--------|--------|---------|
| **السيولة الحالية** | أصول متداولة ÷ خصوم متداولة | قدرة على سداد ديون قصيرة |
| **الربحية** | صافي دخل ÷ إيرادات | كفاءة ربحية |
| **العائد على أصول** | صافي دخل ÷ إجمالي أصول | كفاءة استخدام أصول |
| **الدين إلى حقوق** | إجمالي دين ÷ حقوق ملكية | رافعة مالية |
