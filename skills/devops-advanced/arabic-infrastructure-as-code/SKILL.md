## الهدف

إتقان أساسيات البنية التحتية ككود (IaC).

## أدوات IaC

### ١. Terraform

**المزايا:**
- متعدد السحابات
- حالة مركزية
- Modules قابلة لإعادة استخدام
- Community كبيرة

### ٢. Ansible

**المزايا:**
- بدون وكيل
- سهل تعلم
- Idempotent
- Modules كثيرة

### ٣. CloudFormation

**المزايا:**
- أصلي لـ AWS
- تكامل كامل
- Change Sets
- Stack Policies

## مبادئ IaC

| المبدأ | الوصف |
|--------|-------|
| **Idempotency** | نفس نتيجة بغض النظر مرات تنفيذ |
| **Version Control** | تتبع تغييرات |
| **Testing** | اختبار تغييرات |
| **Documentation** | توثيق بنية |
