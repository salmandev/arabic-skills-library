## الهدف

تقديم إرشادات للصحة النفسية والدعم الأولي باللغة العربية.

## اضطرابات شائعة

### ١. الاكتئاب

**الأعراض:**
```
□ حزن مستمر
□ فقدان اهتمام
□ تغيرات نوم وشهية
□ تعب مستمر
□ أفكار سلبية
```

**المساعدة:**
```
✓ استمع بدون حكم
✓ شجع على علاج متخصص
✓ قدم دعم عملي
✓ تابع بانتظام
```

### ٢. القلق

**الأعراض:**
```
□ قلق مفرط
□ توتر عضلي
□ صعوبة تركيز
□ اضطراب نوم
□ خفقان قلب
```

**المساعدة:**
```
✓ تمارين تنفس
✓ تقنيات استرخاء
✓ تحدٍ أفكار سلبية
✓ روتين منتظم
```

### ٣. الإجهاد (Stress)

**إدارة الإجهاد:**
```
١. تحديد مصادر الإجهاد
٢. تنظيم وقت
٣. تمارين رياضية
٤. نوم كافي
٥. دعم اجتماعي
٦. تقنيات استرخاء
```

## الإسعافات الأولية النفسية

### نموذج الاستماع الفعال

```
١. اقترب بهدوء
٢. اسمع باهتمام
٣. لا تحكم
٤. قدم طمأنينة
٥. شجع على مساعدة متخصصة
٦. تابع الدعم
```

## متى تطلب مساعدة متخصصة

```
□ أفكار انتحارية
□ عدم قدرة على عمل يومي
□ أعراض >٢ أسبوع
□ استخدام مواد
□ أعراض ذهانية
```

## أفضل الممارسات

✅ تحدث بصراحة
✅ قدم دعم غير مشروط
✅ حافظ على سرية
✅ شجع على علاج
✅ اعتنِ بنفسك أولاً

❌ لا تحكم
❌ لا تقل "تفاءل"
❌ لا تستخف بالمشاعر
❌ لا تعد بعلاج
❌ لا تهمل تحذيرات
