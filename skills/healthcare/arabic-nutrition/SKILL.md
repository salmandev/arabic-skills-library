## حالات خاصة

### السكري

**نصائح:**
```
✓ تحكم في كربوهيدرات
✓ اختر كربوهيدرات معقدة
✓ تجنب سكريات بسيطة
✓ توزع وجبات على اليوم
✓ راقب سكر قبل وبعد أكل
```

### الضغط المرتفع

**نصائح:**
```
✓ قلل ملح (<٢٣٠٠ مغ/يوم)
✓ زد بوتاسيوم (موز، بطاطس)
✓ تجنب أطعمة مصنعة
✓ اتبع حمية DASH
```

### الكوليسترول

**نصائح:**
```
✓ قلل دهون مشبعة
✓ تجنب دهون متحولة
✓ زد ألياف قابلة للذوبان
✓ استخدم زيوت صحية
```
