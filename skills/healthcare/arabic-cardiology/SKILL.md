## الهدف

دليل شامل لتشخيص وعلاج أمراض القلب باللغة العربية.

## الفحص السريري

### الأعراض الشائعة

| العرض | الأسباب المحتملة |
|-------|-----------------|
| ألم صدر | ذبحة، نوبة قلبية، ارتجاع |
| ضيق تنفس | فشل قلبي، ربو، انصمام |
| خفقان | اضطراب نظم، قلق، فرط درقية |
| إغماء | اضطراب نظم، صمامات، ضغط |
| تورم أرجل | فشل قلبي، أوردة، كلى |

### تخطيط القلب (ECG)

**الموجات الأساسية:**
- **P wave**: استقطاب أذيني
- **QRS**: استقطاب بطيني
- **T wave**: إعادة استقطاب

**المعدل الطبيعي:**
- ٦٠-١٠٠ نبضة/دقيقة

**الاضطرابات الشائعة:**
- رجفان أذيني (AF)
- تسرع بطيني (VT)
- إحصار قلبي (Heart Block)

## الأمراض القلبية

### ١. مرض الشريان التاجي

**عوامل الخطر:**
- تدخين
- سكري
- ضغط مرتفع
- كوليسترول مرتفع
- تاريخ عائلي

**التشخيص:**
- ECG
- إيكو
- قسطرة قلبية

**العلاج:**
- أسبرين
- ستاتين
- حاصرات بيتا
- رأب وعائي
- جراحة مجازة

### ٢. فشل القلب

**الأعراض:**
- ضيق تنفس
- تعب
- تورم أرجل
- زيادة وزن سريعة

**العلاج:**
- مثبطات ACE
- حاصرات بيتا
- مدرات بول
- تقليل ملح
- تمارين

### ٣. اضطرابات النظم

**الأنواع:**
- بطء نظم (<٦٠)
- تسرع نظم (>١٠٠)
- عدم انتظام

**العلاج:**
- أدوية
- ناظمة قلبية
- صدمة كهربائية

## فحوصات القلب

| الفحص | الاستخدام |
|-------|----------|
| **ECG** | اضطراب نظم، نوبة قلبية |
| **Echocardiogram** | وظيفة ضخ، صمامات |
| **Stress Test** | ذبحة جهدية |
| **Holter** | مراقبة ٢٤ ساعة |
| **Cardiac Cath** | شرايين تاجية |
| **CT Angio** | تصوير شرايين |
