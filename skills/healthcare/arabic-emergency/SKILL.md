## الهدف

دليل شامل للتعامل مع الحالات الطارئة الطبية باللغة العربية.

|-------|---------|-------|
| **A** | مجرى الهواء | Airway |

| **C** | الدورة الدموية | Circulation |

## الحالات الحرجة

### ١. السكتة القلبية

**الأعراض:**
- فقدان الوعي
- عدم التنفس
- عدم النبض

**الإجراء:**
```
١. اطلب الإسعاف فوراً (٩٩٧)
٢. ابدأ الإنعاش القلبي الرئوي (CPR)
٣. استخدم مزيل الرجفان (AED) إن وجد
```

**CPR للبالغين:**
- ٣٠ ضغطة صدرية
- عمق ٥-٦ سم
- سرعة ١٠٠-١٢٠ ضغطة/دقيقة
- تنفسين إنقاذيين

### ٢. النوبة القلبية

**الأعراض:**
- ألم صدري ضاغط
- ألم يمتد للذراع/الفك
- ضيق تنفس
- تعرق بارد
- غثيان

**الإجراء:**
```
١. اتصل بالإسعاف
٢. أعطِ أسبرين ٣٠٠ مغ (إذا لم يكن لديه حساسية)
٣. اجعل المريض في وضع مريح
٤. جهز CPR إذا لزم
```

### ٣. السكتة الدماغية (FAST)

| الحرف | الاختبار | الطبيعي |
|-------|----------|---------|
| **F** | الوجه | اطلب من المريض الابتسام |
| **A** | الذراعين | اطلب رفع الذراعين |
| **S** | الكلام | اطلب تكرار جملة |
| **T** | الوقت | اتصل بالإسعاف فوراً |

### ٤. صدمة الحساسية

**الأعراض:**
- صعوبة تنفس
- تورم وجه/حلق
- طفح جلدي
- دوار

**الإجراء:**
```
١. استخدم قلم الإبينفرين (EpiPen)
٢. اتصل بالإسعاف
٣. اجعل المريض مستلقياً
٤. راقب العلامات الحيوية
```

### ٥. النزيف الحاد

**الإجراء:**
```
١. اضغط مباشرة على الجرح
٢. ارفع الطرف المصاب
٣. استخدم ضمادة ضاغطة
٤. إذا استمر النزيف: اضغط على الشريان
٥. لا تنزع الضمادة المشبعة
```

## قائمة فحص الطوارئ

```
□ مجرى الهواء مفتوح
□ التنفس كافٍ
□ النبض موجود
□ النزيف مسيطر عليه
□ الكسور مثبتة
□ الحروق مغطاة
□ العلامات الحيوية مستقرة
□ الوصول الوريدي موجود
□ الأدوية المعطاة مسجلة
□ المريض مستقر للنقل
```
