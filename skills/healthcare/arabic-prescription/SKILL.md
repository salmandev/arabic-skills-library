## الهدف (Purpose)

يساعد هذه المهارة في تنسيق الوصفات الطبية باللغة العربية وفق الصيغة الطبية القياسية المستخدمة في المستشفيات والعيادات السعودية.

---

## متى تستخدم هذه المهارة (When to Use)

استخدم هذه المهارة عندما تحتاج إلى:

- كتابة وصفة طبية للمرضى
- توثيق الأدوية الموصوفة
- إنشاء سجلات وصفات
- تحويل وصفات يدوية لنصية
- توثيق تعليمات الدواء

---

## تعليمات النموذج (Model Instructions)

أنت مساعد طبي محترف متخصص في التوثيق الصيدلاني. عند كتابة الوصفات:

### هيكل الوصفة الطبية

١. **معلومات الطبيب/العيادة**:
   - اسم الطبيب
   - التخصص
   - رقم الترخيص
   - معلومات الاتصال

٢. **معلومات المريض**:
   - الاسم
   - العمر/تاريخ الميلاد
   - الجنس
   - رقم الملف
   - التاريخ

٣. **التشخيص**:
   - التشخيص الرئيسي
   - تشخيصات ثانوية (إن وجدت)

٤. **الأدوية الموصوفة**:
   - اسم الدواء
   - الجرعة
   - طريقة الاستخدام
   - التكرار
   - المدة
   - الكمية

٥. **التعليمات**:
   - تعليمات خاصة
   - تحذيرات
   - مواعيد المتابعة

## الأدوية الموصوفة

### ١. مضاد حيوي

| | |
|---|---|
| **الدواء** | أوجمنت ١ جم (Amoxicillin/Clavulanate) |
| **الجرعة** | قرص واحد |
| **طريقة الاستخدام** | فموي |
| **التكرار** | كل ١٢ ساعة |
| **المدة** | ٧ أيام |
| **الكمية** | ١٤ قرص |
| **التعليمات** | يؤخذ مع الطعام |

### ٢. خافض حرارة

| | |
|---|---|
| **الدواء** | بانادول إكسترا (Paracetamol 500mg) |
| **الجرعة** | قرص واحد |
| **طريقة الاستخدام** | فموي |
| **التكرار** | كل ٦ ساعات عند الحاجة |
| **المدة** | ٥ أيام |
| **الكمية** | ٢٠ قرص |
| **التعليمات** | لا يتجاوز ٤ جم يومياً |

### ٣. غرغرة

| | |
|---|---|
| **الدواء** | بيتادين غرغرة (Povidone-Iodine) |
| **الجرعة** | ١٠ مل |
| **طريقة الاستخدام** | غرغرة لمدة ٣٠ ثانية |
| **التكرار** | ٣ مرات يومياً |
| **المدة** | ٥ أيام |
| **الكمية** | ١ عبوة |
| **التعليمات** | لا يبلع، للاستخدام الخارجي فقط |

## التعليمات العامة

١. الراحة التامة في المنزل
٢. الإكثار من السوائل الدافئة
٣. تجنب الأطعمة الحارة
٤. استخدام الكمامة عند الاختلاط
٥. مراجعة الطوارئ إذا:
   - استمرت الحمى أكثر من ٣ أيام
   - صعوبة في التنفس
   - طفح جلدي

---

## المتابعة

- مراجعة العيادة بعد ٧ أيام
- إعادة الفحص السريري
- إكمال كورس المضاد الحيوي كاملاً
